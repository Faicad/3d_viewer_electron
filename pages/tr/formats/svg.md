# SVG — Vector

SVG (Scalable Vector Graphics) is an XML-based 2D vector graphics format widely used for web icons, illustrations and UI design.

## Format Info

| Property | Value |
| --- | --- |
| Extensions | `.svg` |
| Category | Vector |
| Render Type | Vector graphics rendering |

## Supported Features

- Vector graphics rendering
- Infinite scaling without quality loss

### General Features

- Drag & drop: drag files directly into the window
- Click to upload: select via file dialog
- Clipboard paste: Ctrl+V after copying
- OrbitControls: rotate / pan / zoom
- Scene tree: hierarchical model structure
- Model export: download as STL or GLB
