# Features

## 🚀 マルチフォーマット対応

STL、GLB、STEP、OBJ、FBX、PLY、3MF など 27 以上の 3D ファイル形式に対応。メッシュ、CAD、点群、ボリュームなどをカバー

## 🎨 PBR レンダリングエンジン

メタルネス/ラフネスワークフロー、環境マップ、マルチライトシステムを備えた Three.js ベースの物理レンダリング

## 🖱️ 豊富なインタラクションツール

OrbitControls カメラ、TransformControls 操作、トポロジ選択（オブジェクト/面/エッジ/頂点）

## 📂 簡単ファイル管理

ドラッグ＆ドロップ、クリップボード貼り付け、自動ディレクトリスキャン、キーボード操作可能なファイルリスト

## 🌓 ダーク/ライトテーマ

ライト、ダーク、システム追従の3つのテーマモードに対応

## 🌐 バイリンガルインターフェース

中国語と英語の完全なUI、手動切り替えまたはシステム言語に自動追従
