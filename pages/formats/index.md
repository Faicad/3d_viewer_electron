# 支持的格式

查看 Faicad 3D Viewer 支持的所有文件格式的详细介绍。

- [STL](stl) — `.stl` — 网格 (Mesh)
- [GLB](glb) — `.glb` — 网格 (Mesh)
- [GLTF](gltf) — `.gltf` — 网格 (Mesh)
- [3MF](3mf) — `.3mf` — 网格 (Mesh)
- [STEP](step) — `.step, .stp` — CAD
- [OBJ](obj) — `.obj` — 网格 (Mesh)
- [PLY](ply) — `.ply` — 网格 (Mesh)
- [FBX](fbx) — `.fbx` — 网格 (Mesh)
- [Collada](dae) — `.dae` — 网格 (Mesh)
- [3DS](3ds) — `.3ds` — 网格 (Mesh)
- [USDZ](usdz) — `.usdz` — 网格 (Mesh)
- [Draco](drc) — `.drc` — 网格 (Mesh)
- [AMF](amf) — `.amf` — 网格 (Mesh)
- [LWO](lwo) — `.lwo` — 网格 (Mesh)
- [3DM](3dm) — `.3dm` — 网格 (Mesh)
- [BVH](bvh) — `.bvh` — 动画 (Animation)
- [MD2](md2) — `.md2` — 动画 (Animation)
- [XYZ](xyz) — `.xyz` — 点云 (Point Cloud)
- [PDB](pdb) — `.pdb` — 点云 (Point Cloud)
- [PCD](pcd) — `.pcd` — 点云 (Point Cloud)
- [VTK](vtk) — `.vtk, .vtp` — 体数据 (Volume)
- [NRRD](nrrd) — `.nrrd` — 体数据 (Volume)
- [GCode](gcode) — `.gcode` — GCode
- [VRML](wrl) — `.wrl` — 其他
- [VOX](vox) — `.vox` — 其他
- [KMZ](kmz) — `.kmz` — 其他
