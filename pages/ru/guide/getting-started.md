# Getting Started

Faicad 3D Viewer is a free and open-source cross-platform 3D model file viewer. It is built on Electron and Three.js, supporting over 27 3D file formats.

Whether you are a 3D printing enthusiast, CAD engineer, or 3D graphics designer, Faicad 3D Viewer provides a smooth 3D model viewing experience.

This guide will help you quickly learn how to install, configure, and use Faicad 3D Viewer.
