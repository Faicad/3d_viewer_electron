# Features

## 🚀 다중 포맷 지원

STL, GLB, STEP, OBJ, FBX, PLY, 3MF 등 27+ 3D 파일 형식을 지원하며 메시, CAD, 포인트 클라우드, 볼륨 등을 포함

## 🎨 PBR 렌더링 엔진

메탈니스/러프니스 워크플로, 환경 맵, 다중 조명 시스템을 갖춘 Three.js 기반 물리 기반 렌더링

## 🖱️ 풍부한 상호작용 도구

OrbitControls 카메라, TransformControls 조작, 토폴로지 선택 (객체/면/모서리/정점)

## 📂 손쉬운 파일 관리

드래그 앤 드롭, 클립보드 붙여넣기, 자동 디렉터리 스캔, 키보드 탐색 가능한 파일 목록

## 🌓 다크/라이트 테마

라이트, 다크, 시스템 설정 연동의 세 가지 테마 모드 지원

## 🌐 이중 언어 인터페이스

중국어와 영어로 된 완전한 UI, 수동 전환 또는 시스템 언어 자동 추적
