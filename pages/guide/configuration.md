# 配置与主题

## 界面设置

点击工具栏的设置按钮（齿轮图标）打开设置面板：

![设置面板](/screenshots/settings-panel.png)

### 主题切换

三种主题模式可选：

| 模式 | 效果 |
|------|------|
| 浅色 | 明亮背景，适合日间使用 |
| 深色 | 暗色背景，适合夜间或低光环境 |
| 跟随系统 | 自动匹配操作系统的颜色模式 |

### 界面语言

支持中文和英文两种界面语言：

- **中文**：完整的简体中文界面
- **英文**：完整的英文界面
- **跟随系统**：自动匹配操作系统语言

切换即时生效，无需重启应用。

### 缩略图预览

可在设置面板中开启或关闭文件列表中的缩略图预览功能。

## 显示模式

视口工具栏提供四种显示模式：

| 模式 | 说明 |
|------|------|
| 实体 | 默认模式，完整 PBR 材质渲染 |
| 线框 | 显示三角网格线框 |
| 实体+线框 | 实体表面叠加线框 |
| 网格 | 显示参考网格辅助线 |

## 环境贴图

在环境面板中可以：

- 从三套预设环境贴图中切换
- 加载自定义 `.hdr` 或 `.exr` 环境贴图
- 环境贴图影响模型的反射和光照效果

## 全屏模式

点击工具栏的全屏按钮或按 `F11`（如果系统支持）进入全屏模式，获得更大的 3D 视口空间。按 `Esc` 退出全屏。
