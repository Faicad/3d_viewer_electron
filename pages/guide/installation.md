# Installation

## Download

Download the latest release from the [GitHub Releases page](https://github.com/faicad/3d_viewer_electron/releases).

### Windows

| File | Description |
|------|-------------|
| `3D-Model-Viewer-Setup-x.x.x.exe` | NSIS installer with desktop & start menu shortcuts |
| `win-unpacked.zip` | Portable version, unzip and run `3D_Viewer.exe` |

### macOS

| File | Description |
|------|-------------|
| `3D-Model-Viewer-x.x.x.dmg` | Standard DMG image, drag to Applications |

### Linux

| File | Description |
|------|-------------|
| `3D-Model-Viewer-x.x.x.AppImage` | Self-contained AppImage |
| `3D-Model-Viewer-x.x.x.deb` | Debian/Ubuntu package |

## System Requirements

- **Windows**: Windows 10/11 (x64)
- **macOS**: macOS 12+ (Intel & Apple Silicon)
- **Linux**: Modern distributions (x64)

No additional runtime or libraries required.

## Updates

Check the [Releases page](https://github.com/faicad/3d_viewer_electron/releases) for the latest version.
