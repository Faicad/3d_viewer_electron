# NRRD — Volume

NRRD (Nearly Raw Raster Data) is a volume data format in medical and scientific imaging, supporting multi-dimensional raster data.

## Format Info

| Property | Value |
| --- | --- |
| Extensions | `.nrrd` |
| Category | Volume |
| Render Type | Volume rendering |

## Supported Features

- Volume rendering
- Proxy cube display

### General Features

- Drag & drop: drag files directly into the window
- Click to upload: select via file dialog
- Clipboard paste: Ctrl+V after copying
- OrbitControls: rotate / pan / zoom
- Scene tree: hierarchical model structure
- Model export: download as STL or GLB
