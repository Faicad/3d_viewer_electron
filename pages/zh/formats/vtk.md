# VTK — 体数据 (Volume)

VTK (Visualization Toolkit) 是科学可视化领域的标准数据格式，支持多种数据类型。

## 基本信息

| 属性 | 值 |
| --- | --- |
| 扩展名 | `.vtk, .vtp` |
| 分类 | 体数据 (Volume) |
| 渲染方式 | 标准三角网格渲染 |

## 支持的特性

- PBR 材质渲染
- 标准三角网格显示

### 通用功能

- 拖拽加载：直接将文件拖入应用窗口
- 点击上传：通过文件对话框选择
- 剪贴板粘贴：复制文件后 Ctrl+V
- OrbitControls：旋转 / 平移 / 缩放
- 场景树：层次化展示模型结构
- 模型导出：下载为 STL 或 GLB
