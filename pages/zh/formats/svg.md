# SVG — 矢量 (Vector)

SVG (Scalable Vector Graphics) 是基于 XML 的二维矢量图形格式，广泛应用于 Web 图标、插图和 UI 设计。

## 基本信息

| 属性 | 值 |
| --- | --- |
| 扩展名 | `.svg` |
| 分类 | 矢量 (Vector) |
| 渲染方式 | 矢量图形渲染 |

## 支持的特性

- 矢量图形渲染
- 无限缩放不失真

### 通用功能

- 拖拽加载：直接将文件拖入应用窗口
- 点击上传：通过文件对话框选择
- 剪贴板粘贴：复制文件后 Ctrl+V
- OrbitControls：旋转 / 平移 / 缩放
- 场景树：层次化展示模型结构
- 模型导出：下载为 STL 或 GLB
