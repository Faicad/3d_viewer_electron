# BVH — 动画 (Animation)

BVH (Biovision Hierarchy) 是生物运动捕捉数据格式，以骨架层次结构渲染动画。

## 基本信息

| 属性 | 值 |
| --- | --- |
| 扩展名 | `.bvh` |
| 分类 | 动画 (Animation) |
| 渲染方式 | 骨骼动画渲染 |

## 支持的特性

- 骨架动画渲染
- 骨骼层次展示

### 通用功能

- 拖拽加载：直接将文件拖入应用窗口
- 点击上传：通过文件对话框选择
- 剪贴板粘贴：复制文件后 Ctrl+V
- OrbitControls：旋转 / 平移 / 缩放
- 场景树：层次化展示模型结构
- 模型导出：下载为 STL 或 GLB
