# XYZ — 点云 (Point Cloud)

XYZ 是简单的点坐标数据格式，每行包含 X/Y/Z 坐标，以点云方式渲染。

## 基本信息

| 属性 | 值 |
| --- | --- |
| 扩展名 | `.xyz` |
| 分类 | 点云 (Point Cloud) |
| 渲染方式 | 点云渲染 |

## 支持的特性

- 点云渲染方式
- 每点颜色渲染（如包含）

### 通用功能

- 拖拽加载：直接将文件拖入应用窗口
- 点击上传：通过文件对话框选择
- 剪贴板粘贴：复制文件后 Ctrl+V
- OrbitControls：旋转 / 平移 / 缩放
- 场景树：层次化展示模型结构
- 模型导出：下载为 STL 或 GLB
