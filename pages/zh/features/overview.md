# Features

## 🚀 多格式支持

支持 STL、GLB、STEP、OBJ、FBX、PLY、3MF 等 27+ 种 3D 文件格式，覆盖网格、CAD、点云、体数据等类别

## 🎨 PBR 渲染引擎

基于 Three.js 的物理渲染系统，支持金属度/粗糙度工作流、环境贴图和多光源系统

## 🖱️ 丰富的交互工具

OrbitControls 相机控制、TransformControls 变换工具、拓扑选择（物体/面/边/顶点）

## 📂 便捷的文件管理

拖拽加载、剪贴板粘贴、目录自动扫描、键盘导航的文件列表，快速切换模型

## 🌓 深色/浅色主题

支持浅色、深色和跟随系统三种主题模式，适应不同使用环境

## 🌐 中英双语界面

完整的中文和英文界面支持，可手动切换或跟随系统语言自动切换
