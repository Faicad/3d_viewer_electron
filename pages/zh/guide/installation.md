# 安装与构建

## 下载方式

从 [GitHub Releases 页面](https://github.com/faicad/3d_viewer_electron/releases) 下载对应平台的安装包。

### Windows

| 文件 | 说明 |
|------|------|
| `3D-Model-Viewer-Setup-x.x.x.exe` | NSIS 安装包，安装后桌面和开始菜单创建快捷方式 |
| `win-unpacked.zip` | 绿色免安装版，解压后直接运行 `3D_Viewer.exe` |

### macOS

| 文件 | 说明 |
|------|------|
| `3D-Model-Viewer-x.x.x.dmg` | 标准 DMG 镜像，拖入 Applications 文件夹 |

### Linux

| 文件 | 说明 |
|------|------|
| `3D-Model-Viewer-x.x.x.AppImage` | 直接运行的 AppImage 格式 |
| `3D-Model-Viewer-x.x.x.deb` | Debian/Ubuntu 安装包 |

## 系统要求

- **Windows**：Windows 10/11（x64）
- **macOS**：macOS 12+（Intel 和 Apple Silicon）
- **Linux**：主流发行版（x64）

无需安装额外的运行时或依赖库。

## 从源码构建（仅供开发者）

如果你需要自行构建，请参考 [GitHub 仓库](https://github.com/faicad/3d_viewer_electron) 的 README。

## 更新

- 新版本发布后，打开应用会提示更新
- 也可以定期查看 [Releases 页面](https://github.com/faicad/3d_viewer_electron/releases) 获取最新版本
